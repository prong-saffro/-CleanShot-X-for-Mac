## Key Features

### Screenshot Capture

CleanShot X allows users to take:

* Full-screen screenshots
* Selected area screenshots
* Window-specific captures
* Scrolling screenshots (entire webpages or long content)

### Screen Recording

Record your screen in high quality with options for:

* Full screen recording
* Selected region recording
* Webcam overlay
* Audio recording (system + microphone)

### Annotation Tools

Edit screenshots instantly with tools such as:

* Arrows and shapes
* Text labels
* Blur and pixelate (for privacy)
* Highlights and markers

### Quick Access Overlay

A floating capture menu allows instant access to tools without opening the full app.

### Cloud Upload Support

Automatically upload screenshots and recordings to cloud storage or shareable links.

### Built-in Screenshot Library

Organize all captures in a clean library with tagging and search options.

### Self-Timer and Delay Capture

Take screenshots after a delay to capture menus, tooltips, or dynamic content.

### Background Removal (AI Feature)

Remove backgrounds from images quickly for presentations or design work.
